Connect Your Micro Oven To Internet

Pre-Requisites for Smart Connection

Turn "On" the device and Select "Menu" option

Select "More" option

Select "Home Connect Assistent"

Automatic network connection" prompt message arrives on display

Turn On "WPS" Button

"Network connection successful" message on Appliance indicating connection is established.

Select "Connect to app"

"Confirm Connection on the mobile device" prompt message arrives on display

Select "connect" on Launch of your App

Select "Continue"

Device Type "Microwave" is added to your mobile.
